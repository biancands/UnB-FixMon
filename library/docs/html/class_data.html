Descrição detalhada

Representa uma data no sistema.

Regras de formato:

  • A data deve seguir o formato DD-MM-AAAA, onde DD é o dia (00 a 31), MM é o mês (01 a 12) e AAAA é o ano (2000 a 2100).
  • Leva em consideração anos bissextos.
