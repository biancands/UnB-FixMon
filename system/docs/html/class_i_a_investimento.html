Descrição detalhada

Interface para operações de execução relacionadas a investimentos.

Esta interface define o contrato para a execução de operações relacionadas a investimentos, utilizando o CPF como identificador. É implementada pela controladora CntrIAInvestimento, que então delega o serviço para a interface de serviço correspondente.
