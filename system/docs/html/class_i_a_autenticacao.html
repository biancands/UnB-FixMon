Descrição detalhada

Interface para operações de autenticação de usuários.

Esta interface define o contrato para operações de autenticação de usuários, utilizando CPF e senha como identificadores. É implementada pela controladora CntrIAAutenticacao, que então delega o serviço para a interface de serviço ISAutenticacao.
