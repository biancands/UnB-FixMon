Descrição detalhada

Interface para operações CRUD em títulos e pagamentos.

Esta interface define o contrato para operações CRUD em títulos e pagamentos, ela é utilizada pelas controladoras da camada de apresentação para intermediar as operações entre o usuário e os dados persistidos.
