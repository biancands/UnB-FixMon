Descrição detalhada

Interface para operações de autenticação de usuários.

Esta interface define o contrato para a operação de autenticação de usuários, ela é utilizada pelas controladoras da camada de apresentação para intermediar as operações de autenticação entre o usuário e os dados persistidos.
