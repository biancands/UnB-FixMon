Descrição detalhada

Interface para operações de criação e execução em contas.

Esta interface define o contrato para operações de criação de contas e execução de ações relacionadas a contas, utilizando o CPF como identificador. É implementada pela controladora CntrIAConta, que então delega o serviço para a interface de serviço ISConta.
